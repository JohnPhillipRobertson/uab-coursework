## CS 332 Lab 9

This program extends execvp.c by handling signals.

## Author
John Robertson

## Getting started

To compile:
```bash
gcc lab9.c
```

## Usage


```bash
$ ./a.out ./<command> [args]
```
Once it executes the given command, SIGTSTP or SIGINT can be sent to the process without killing the parent.
