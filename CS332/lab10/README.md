## CS 332 Lab 10

This program extends lab 7 by replacing stdout and stderr with corresponding files per each process.

## Author
John Robertson

## Getting started

To compile:
```bash
make
```

To run:
```
lab10 <commands_file>
```

## Usage


```bash
lab10 <commands_file>
```
The "commands file" should contain Linux commands, separated by newlines and with no space after them.
