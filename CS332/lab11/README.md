## CS 332 Lab 11

This program is a mock shell laid over the bash shell.

## Author
John Robertson

## Getting started

To compile:
```bash
make
```

To run:
```
lab11
```

## Usage


```bash
lab11
```
Will cause the shell to start. From there, any chain of Linux commands can be entered, with any combination of pipes and arrows. To quit, enter `quit`.
