## CS 332 Lab 12

This program calculates the sum of integers to *n* in a parallel manner.

## Author
John Robertson

## Getting started

To compile:
```bash
make
```
or
```bash
gcc -o lab12 -lpthread lab12.c
```

To run:
```
lab12 <int> <int>
```

## Usage


```bash
lab12 <int> <int>
```
Will give the sum of integers from 1 to the first argument. The second argument provides how many threads the calculation is to be done across.
